# ROI Analyzer 계산 공식 문서

> 이 문서는 `roi_analyzer_merged.jsx`에서 사용되는 모든 핵심 계산 공식을 정리한 것입니다.

---

## 1. 채널별 지표 계산 (`calcChannelMetrics`)

각 판매 채널(쿠팡, 네이버 등)의 개별 수익성을 계산합니다.

```
# 수수료
수수료(commission) = 매출(revenue) × 수수료율(commissionRate)
  - 쿠팡 기본 수수료율: 10.8%

# 매출원가 (COGS)
매출원가(cogs) = 단가(unitCost) × 판매수량(unitsSold)

# 영업이익
영업이익(opProfit) = 매출 - 매출원가 - 광고비 - 수수료 - 물류비 - 반품비용 - 기타비용

# 영업이익률
영업이익률(opMargin) = (영업이익 / 매출) × 100  [%]
  - 매출이 0이면 0%

# ROAS (광고수익률)
ROAS = (매출 / 광고비) × 100  [%]
  - 광고비가 0이면 0%
  - 200% 이상이면 손익분기 이상 (광고비 대비 2배 매출)
```

## 2. 상품 종합 지표 계산 (`calcProductMetrics`)

모든 채널의 데이터를 합산하여 상품 단위의 KPI를 산출합니다.

### 2.1 기본 합산 지표

```
총매출(totalRevenue)     = 모든 채널 매출 합계
총광고비(totalAdSpend)   = 모든 채널 광고비 합계
총수수료(totalCommission)= 모든 채널 수수료 합계
총물류비(totalLogistics)  = 모든 채널 물류비 합계
총반품비(totalReturnCost) = 모든 채널 반품비용 합계
총매출원가(totalCogs)     = 모든 채널 COGS 합계
총기타비용(totalOtherCost)= 모든 채널 기타비용 합계
총판매수량(totalUnitsSold) = 모든 채널 판매수량 합계
총리뷰수(totalReviews)    = 모든 채널 리뷰수 합계
```

### 2.2 영업이익 및 이익률

```
# 총 영업이익
총영업이익(totalOpProfit) = 총매출 - 총매출원가 - 총광고비 - 총수수료 - 총물류비 - 총반품비 - 총기타비용

# 영업이익률
영업이익률(opMargin) = (총영업이익 / 총매출) × 100  [%]

# 통합 ROAS
통합ROAS(combinedRoas) = (총매출 / 총광고비) × 100  [%]
```

### 2.3 ROI (투자수익률)

```
# 투자금
투자금(investment) = 사용자 입력 투자금(investmentOverride) || 기본 투자금(investment)
  - 투자금 = 단가(unitCost) × 수입수량(importQty) (기본값)

# 기간 ROI
기간ROI(periodROI) = (총영업이익 / 투자금) × 100  [%]
  - 설정된 분석기간(예: 180일) 동안의 수익률

# 연환산 ROI
연환산ROI(annualROI) = 기간ROI × (365 / 분석기간일수)  [%]
  - 기간 ROI를 365일 기준으로 환산
  - 예) 180일 기간 ROI가 50%이면 → 연환산 ROI = 50% × (365/180) ≈ 101.4%
```

### 2.4 재고회전율 및 판매속도

```
# 일 판매량
일판매량(dailySales) = 총판매수량 / 분석기간일수

# MOQ 소진 일수
MOQ소진일수(daysToSellMOQ) = 수입수량(importQty) / 일판매량
  - 일판매량이 0이면 무한대(Infinity)

# 연간 재고회전율
연간회전율(annualTurnover) = 365 / MOQ소진일수
  - MOQ소진일수가 0이거나 무한대이면 0
  - 높을수록 재고가 빠르게 회전 (좋은 지표)

# 투자금 회수일
회수일(paybackDays) = 투자금 / 일영업이익
  - 일영업이익 = 총영업이익 / 분석기간일수
  - 일영업이익이 0 이하이면 null (회수 불가)

# 자본회전율
자본회전율(capitalTurnover) = 총매출 / 투자금
```

### 2.5 투자점수 (Investment Score)

```
투자점수(investmentScore) = min(100, max(0, round((연환산ROI / 160) × 100)))

# 해석:
#   - 연환산 ROI 160% → 투자점수 100점 (만점)
#   - 연환산 ROI 80%  → 투자점수 50점
#   - 연환산 ROI 0%   → 투자점수 0점
#   - 0~100 범위로 클램핑
```

### 2.6 가중 평균 지표

```
# 가중평균 반품률 (판매수량 기준 가중)
가중반품률(weightedReturnRate) = Σ(채널반품률 × 채널판매수량) / 총판매수량

# 가중평균 평점 (리뷰수 기준 가중)
가중평점(weightedRating) = Σ(채널평점 × 채널리뷰수) / 총리뷰수

# 평균 전환율
평균전환율(weightedCvr) = Σ(채널전환율) / 채널수
```

### 2.7 투자 상태 판정 (Investment Status)

```
# 기본 판정 로직 (투자점수 기반)
if 투자점수 >= 75 AND 연환산ROI >= 100% AND 이익률 >= 15%:
    상태 = "확대" (expand)
elif 투자점수 >= 50:
    상태 = "유지" (maintain)
elif 투자점수 >= 25:
    상태 = "축소" (reduce)
else:
    상태 = "중단" (stop)

# 반품률 보정
if 가중반품률 > 10% AND (상태가 "확대" 또는 "유지"):
    상태 = "축소"로 강등

# 적자 지속 보정
if 총영업이익 < 0 AND 모든 주간 영업이익이 음수:
    상태 = "중단"으로 강등
```

### 2.8 손익분기 ROAS (BE-ROAS)

```
# 평균 마진율
평균마진율(avgMarginRate) = (총매출 - 총매출원가) / 총매출

# 평균 수수료율
평균수수료율(avgCommRate) = 총수수료 / 총매출

# 손익분기 ROAS
BE-ROAS(beRoas) = (1 / (평균마진율 - 평균수수료율)) × 100  [%]
  - 광고비를 지출해도 손해 보지 않는 최소 ROAS
  - (마진율 - 수수료율)이 0 이하이면 0
```

## 3. 포트폴리오 건강성 점수 (`healthScore`)

대시보드에서 전체 포트폴리오의 건강 상태를 0~100점으로 평가합니다.

```
건강성점수 = min(100, max(0, 합산값))

합산 항목:
  +25점: 총영업이익 > 0 (흑자)
  +20점: 평균ROI > 50%  (또는 +10점: 평균ROI > 0%)
  +20점: 평균ROAS > 200% (또는 +10점: 평균ROAS > 100%)
  +20점: 평균반품률 < 5%  (또는 +10점: 평균반품률 < 10%)
  +15점: 위험상품 비율 < 10% (또는 +8점: 위험상품 비율 < 30%)

# 위험상품 = 상태가 "축소" 또는 "중단"인 상품
```

## 4. What-If 시뮬레이션

가격, 광고비, 반품률 변동에 따른 예상 실적을 시뮬레이션합니다.

### 4.1 판매량 변동 모델

```
# 가격 탄력성: 가격 1% 변동 → 판매량 -1.1% 변동 (역방향)
가격변동승수(unitMult) = 1 + (가격변동% / 100) × (-1.1)
  - 예) 가격 10% 인상 → 판매량 11% 감소

# 광고 효과: 광고비 1% 변동 → 판매량 +0.3% 변동 (같은 방향)
광고변동승수(adMult) = 1 + (광고비변동% / 100) × 0.3
  - 예) 광고비 50% 증가 → 판매량 15% 증가

# 예상 판매수량
예상판매수량(newUnits) = max(0, round(기존판매수량 × 가격변동승수 × 광고변동승수))
```

### 4.2 예상 수익 계산

```
# 예상 판매가
예상판매가(newPrice) = 기존판매가 × (1 + 가격변동% / 100)

# 예상 매출
예상매출(newRev) = 예상판매수량 × 예상판매가

# 예상 광고비
예상광고비(newAd) = 기존광고비 × (1 + 광고비변동% / 100)

# 예상 반품률
예상반품률(newReturnRate) = max(0, 기존반품률 + 반품률변동값)

# 예상 ROAS
예상ROAS(newRoas) = (예상매출 / 예상광고비) × 100

# 예상 물류비
예상물류비 = 예상판매수량 × 개당물류비

# 예상 매출원가
예상매출원가 = 예상판매수량 × 단가

# 예상 수수료
예상수수료 = 예상매출 × 평균수수료율

# 예상 반품비용
예상반품비용 = 예상매출 × (예상반품률 / 100)

# 예상 영업이익
예상영업이익(newProfit) = 예상매출 - 예상매출원가 - 예상광고비 - 예상수수료 - 예상물류비 - 예상반품비용

# 예상 이익률
예상이익률(newOpMargin) = (예상영업이익 / 예상매출) × 100

# 예상 ROI
예상기간ROI(newPeriodROI) = (예상영업이익 / 투자금) × 100
예상연환산ROI(newAnnualROI) = 예상기간ROI × (365 / 분석기간일수)

# 예상 투자점수
예상투자점수(newScore) = min(100, max(0, round((예상연환산ROI / 160) × 100)))
```

### 4.3 목표 ROAS 기반 광고 예산

```
# 목표 광고비 역산
목표광고비(targetAdBudget) = 예상매출 / (목표ROAS / 100)
  - 목표 ROAS를 달성하려면 이 금액 이하로 광고비를 써야 함
```

## 5. 마케팅 비용 분석 (`mktAnalysis`)

쿠팡 광고 외 추가 마케팅 비용(인플루언서, SNS 등)을 포함한 종합 분석입니다.

### 5.1 카테고리별 마케팅 지표

```
# 카테고리별 ROAS
카테고리ROAS = (예상매출 / 지출비용) × 100  [%]

# 카테고리별 ROI
카테고리ROI = ((예상매출 - 지출비용) / 지출비용) × 100  [%]

# 이익 기여도
이익기여도(profitContrib) = 예상매출 - 지출비용

# 비용 점유율
비용점유율(costShare) = (카테고리지출 / 전체마케팅지출) × 100  [%]
  - 전체마케팅지출 = 쿠팡광고비 + 추가마케팅비용 합계
```

### 5.2 마케팅 반영 조정 지표

```
# 추가 마케팅 비용을 반영한 조정 지표
조정영업이익(adjProfit) = 기존영업이익 - 추가마케팅비용 합계
조정이익률(adjOpMargin) = (조정영업이익 / 총매출) × 100
조정기간ROI(adjPeriodROI) = (조정영업이익 / 투자금) × 100
조정연환산ROI(adjAnnualROI) = 조정기간ROI × (365 / 분석기간일수)
```

## 6. 매출 예측 (`ForecastPage`)

과거 실적 기반으로 미래 매출과 이익을 예측합니다.

### 6.1 성장률 및 매출 예측

```
# 일평균 매출
최근일매출(recentDaily) = 총매출 / 분석기간일수

# 이전 일매출 (추정)
이전일매출(prevDaily) = 최근일매출 × 0.9
  - 현재 매출이 이전 대비 약 11% 성장했다고 가정

# 성장률
성장률(growthRate) = ((최근일매출 - 이전일매출) / 이전일매출) × 100  [%]
  - 기본 모델에서는 약 11.1%로 고정

# 3개월 예측 매출
3개월예측매출(forecast3m) = 최근일매출 × (1 + 성장률/100) × 90

# 6개월 예측 매출
6개월예측매출(forecast6m) = 최근일매출 × (1 + 성장률/100) × 180

# 예측 이익
3개월예측이익(forecastProfit3m) = 3개월예측매출 × (이익률 / 100)
6개월예측이익(forecastProfit6m) = 6개월예측매출 × (이익률 / 100)
```

### 6.2 투자 효율성 점수

```
효율성점수(efficiency) = 연환산ROI × 연간회전율
  - 회전율이 0이면 0.1로 대체
  - ROI가 높고 회전이 빠른 상품일수록 높은 점수
```

### 6.3 리스크 점수

```
리스크점수(riskScore) = 반품위험 + ROI위험 + ROAS위험  (최대 100)

# 반품 위험
  반품률 > 10% → +30점
  반품률 > 5%  → +15점
  반품률 <= 5% → +0점

# ROI 위험
  연환산ROI < 0%  → +40점
  연환산ROI < 30% → +20점
  연환산ROI >= 30% → +0점

# ROAS 위험
  ROAS < 100% → +30점
  ROAS < 200% → +10점
  ROAS >= 200% → +0점
```

## 7. 상품 데이터 초기화 (`PRODUCTS`)

원시 데이터에서 상품 객체를 생성할 때의 마진율 계산입니다.

```
# 마진율 추정 (주간 트렌드 생성용)
마진율(marg) = max(0.05, 1 - 단가/판매가 - 0.108 - 0.05 - 반품률/100)

# 구성요소:
#   1 (매출 100%)
#   - 단가/판매가     : 원가율
#   - 0.108           : 쿠팡 수수료율 (10.8%)
#   - 0.05            : 물류비 추정 비율 (5%)
#   - 반품률/100      : 반품 손실률
#   최소값 5%로 클램핑

# 주간 매출 시뮬레이션 (6개월 = 6개 월간 데이터)
월별가중치 = [0.85, 0.80, 0.95, 1.0, 1.1, 1.05]
월매출 = round(총매출 / 6 × 월별가중치)
월이익 = round(월매출 × 마진율)
월광고비 = round(총광고비 / 6 × 월별가중치)
월ROAS = 월광고비 > 0 ? round(월매출 / 월광고비 × 100) : 0
```

## 8. 금액 표시 포맷 (`fmt`, `fmt2`)

```
# fmt (메인 테이블용)
  1억 이상: "1.3억원" 형태
  1억 미만: "3,147,000원" 형태 (콤마 구분)

# fmt2 (카드/요약용)
  1억 이상: "1.3억"
  1만 이상: "3147만"
  1만 미만: "9,500" (콤마 구분)

# fmtPct (퍼센트)
  소수점 1자리 + "%"  예) "24.0%"
```

## 9. 주요 기준값 요약

| 지표 | 기준값 | 의미 |
|------|--------|------|
| 투자점수 만점 기준 | 연환산 ROI 160% | 100점 환산 기준 |
| 확대 판정 | 점수 >= 75, ROI >= 100%, 이익률 >= 15% | 3가지 모두 충족 |
